== Description ==

WP User Registration is a plugin for change Wordpress original registration interface for users to register their choice passwords and other item(like aim, gtalk, nickname and so on), and no longer produced by the system password and sent to mail, and an increase in the registration process of validation code to prevent malicious registration.

For screenshots, please visit the <a href="http://blog.2i2j.com/plugins/wp-user-registration">official plugin page</a>.

<h3>Admin Features</h3>
<ul>
<li>Admin and Users Panel Options</li>
<li>Can choose using validation code or not</li>
<li>Can choose some other item you want to input on register</li>
</ul>

<h3>User Features</h3>
<ul>
<li>Choose own passwords when register.</li>
<li>Must input validation code to prevent malicious registration.</li>
</ul>

== Installation ==

1. Just unzip
2. upload the "wp-user-registration" folder to your '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Customize the validation code in the 'Option' menu in Wordpress.

== Usage ==

For admin...

You can edit global admin options by using the Options->WP User Registration panel.

For anonymous...

If you are a user, you can choose your own passwords in register replace for random passwords.
